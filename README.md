## 🧪 เริ่มต้นพัฒนา

ติดตั้ง dependencies:

```bash
bun install
```

รัน development server:

```bash
bun run dev
```

build production:

```bash
bun run build
```

ถ้าต้องการใช้ npm ก็สามารถใช้ `npm install`, `npm run dev` และ `npm run build` ได้เช่นกัน
